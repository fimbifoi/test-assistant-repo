## Installation Instructions
1. Clone the repository:
    ```bash
    git clone https://github.com/fimbifoi/test-assistant-repo.git
    ```
2. Navigate to the repository directory:
    ```bash
    cd test-assistant-repo
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
